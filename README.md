# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/thorfinnvessel/OIBSIP.git
cd OIBSIP
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```
